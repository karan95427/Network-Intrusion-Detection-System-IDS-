# Network Intrusion Detection System (IDS)

A Python-based Network Intrusion Detection System that captures live network packets and analyzes traffic patterns to detect suspicious activity such as port scanning.

This project was built to understand how real-world network security tools monitor traffic and detect abnormal behavior.

---

## Features

• Live packet sniffing using Scapy  
• Protocol detection (TCP / UDP)  
• Port monitoring and traffic analysis  
• Port scan detection based on suspicious port access patterns  
• Attack alert system with logging  
• Top Talkers detection to identify high traffic hosts  
• Real-time monitoring of network packets  

---

## Technologies Used

- Python
- Scapy
- Networking Fundamentals (TCP/IP, Ports, Protocols)

---

## How It Works

The IDS monitors network traffic and analyzes packet information.
